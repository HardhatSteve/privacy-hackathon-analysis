# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

### ✅ Expected Behavior
- Technical, constructive discussions
- Respectful disagreement
- Focus on code quality and security
- Helping newcomers learn

### ❌ Unacceptable Behavior
- Harassment or discrimination
- Personal attacks
- Spam or off-topic content
- Sharing private information
- Discussion of illegal activities

## Scope

This Code of Conduct applies to:
- GitHub Issues and PRs
- Code comments
- Documentation
- All project spaces

## Enforcement

1. **Warning**: First violation
2. **Temporary Ban**: Repeated violations
3. **Permanent Ban**: Severe or continued violations

## Reporting

Report violations via:
- GitHub: Flag the content
- Email: conduct@privacyexecutionlayer.org

Reports are confidential.

## Attribution

Based on the [Contributor Covenant](https://www.contributor-covenant.org/).
